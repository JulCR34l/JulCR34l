## 📌 Featured Projects

### 🎓 U•link — University Social Platform (PATRICIA)
> A full-stack social network built exclusively for ECI students, featuring real-time communication, AI matching, gamification and student wellness tools.

- **Frontend:** React 18 + TypeScript + Vite + Tailwind CSS — OTP registration, real-time group chats, collaborative canvas, voice rooms, integrated board game and AI wellness chatbot.
- **Communication Backend:** Spring Boot microservice handling WebSockets (STOMP), DM and group messaging, WebRTC voice signaling, multi-node sync via Redis Pub/Sub and RabbitMQ event integration.
- **Matching Backend:** Spring Boot microservice with hexagonal architecture that calculates student compatibility (interests, academic affinity, availability), manages match queues in Redis and publishes confirmed matches via RabbitMQ.

`React` `TypeScript` `Java` `Spring Boot` `Redis` `PostgreSQL` `RabbitMQ` `Docker` `WebSockets` `Microservices`
